Inside Your Genome


Genotyping for the Inside Your Genome project is underway, and it's time to prepare for the return of results. We invite you to contribute to the project by helping to write text for the description of the traits and associated SNPs being genotyped as part of the project.

Trait and SNP Descriptions

Login information

If you wish to view or edit trait and SNP descriptions for the Inside Your Genome project, you will be asked to login using Shibboleth (with support for EBI, Hinxton Hall, and Sanger logins). Please contact hgi@sanger.ac.uk if you experience issues with login.

Trait Categories

After login, you will be presented with the list of trait categories included in the Inside Your Genome project, each of which can be expanded to show the traits included in the category, along with the current description text for that trait and a list of SNPs that are included on the Inside Your Genome genotyping panel (each of which also has a description field). Initially, these descriptions are populated only with placeholder text (e.g. "trait_description_EYE"), as we are hoping members of the community will help the project to provide descriptions of each trait and SNP. These descriptions will be used to augment the results returned to each study participant (which will be done anonymously on a separate web site) with background information that may be of interest given their genotyping results.

Editing trait and SNP descriptions

You are invited to edit the descriptions to provide background information about each trait and SNP in the project. To do so, simply click the "Edit" button next to the trait you wish to edit, and use the in-browser real-time collaborative editor to edit the description. A persistent chat is also included and is attached to each description -- you can use that to coordinate efforts between multiple editors working simultaneously on the same text, to leave notes about work you have partially completed that someone else might want to take up (though you could also include that in the text), or for anything else you can think of. Don't worry about deleting or changing someone else's work -- the full history of the editing session is saved (and accessible via the interface), so you (or anyone else) can go back and retrieve text that has been deleted if you've decided it's useful after all.

Adding Citations

In the description text you are encouraged to include references to support your statements, and to make that easier we support two shorthand formats for pubmed id (PMID) and digital object identifiers (DOI) in addition to standard URI (e.g. http/https) links. To cite a particular paper indexed by pubmed, you can simply write "pmid:" followed by the pubmed id (e.g. pmid:11237011). This will automatically be linked to the pubmed entry on the NIH web site. Likewise, to cite a DOI, enter "doi:" followed by the DOI itself (e.g. 10.1038/35057062), which will create a link directly to the identified document.
